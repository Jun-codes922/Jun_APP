###### <3.2변수>

1. 변수는 값을 담아두는 공간을 말한다. 

2\. 숫자,문자열, 불리언 값 등을 다양한 자료형의 데이터를 변수에 담아 관리할 수 있다.



다트에서 변수의예시

1. 빈 박스 : 변수자체를 의미한다. 변수는 값을 저장하는 공간이다.
2. 'author'라는 적힌 포스트잇: 변수의 의미를 의미한다.
3. '개발하는'남자'라는 문자 : 변수에 저장된 값을 의미한다. 변수는 다양한 데이터타입의 값을 저장할 수 있다.

###### <연산자>

산술 연산자

print(5+3); // 8  더하기

print(10-4); // 6  빼기

print(3\*4); // 12  곱하기

print(15/3); // 5  나누기

print(17%5); // 2



비교 연산자

print(5 > 3); // true

print(5 <= 3); // false

print(5 == 5); // true
