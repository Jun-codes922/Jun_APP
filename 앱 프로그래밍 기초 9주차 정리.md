##### 4.4.6 SizedBox 위젯

###### SizedBox 위젯은 고정된 크기의 상자를 만드는데 사용되고

###### 위젯 사이에 간격(padding)을 추가하거나 레이아웃에 빈 공간을 만드는데 사용한다



* ###### 두 개의 Cotainer사이에 간격을 만들고 싶다면 하나의 Container에 마진을 지정할 수 도 있지만

###### &nbsp;   보통 이런 경우에 SizedBox를 사용하여 간격을 조절한다

* ###### SizedBox를 사용하면 더 간편하게 원하는 크기의 간격을 만들 수 있는데 다음과 같이 사용할 수 있다



&nbsp;Column(

&nbsp;    mainAxisAlignment: MainAxisAlinment.cneter,

&nbsp;children: \[

&nbsp;   Container(

&nbsp;	 color: Colors.red,	

&nbsp;	 width: 100

&nbsp;	 height: 40,

&nbsp;    ),

&nbsp;    const SizeBox(height: 10),

&nbsp;    Container(

&nbsp;	 color: Colors.blue,

&nbsp;        width: 100,

&nbsp;	 height: 40,

&nbsp;    ),

&nbsp;],

),





###### Row에서는 height가 아닌 width를 사용하여 가로 간격을 지정할 수 있다

&nbsp; Row(

     mainAxisAlignment: MainAxisAlinment.cneter,

 ],

##### 4.4.7 ListView 위젯

###### ListView 위젯은 스크롤할 수 있는 목록을 표시하는데 사용된다.

###### 가로 또는 세로로 스크롤할 수 있는 긴 항목 목록을 만들때 유용하다



scrollDirection

1. 이 옵션은 Axis enum 타입의 값을 지정해야 한다
2. 가로 스크롤을 원할 때는 Axis.horizontal을,세로 스크롤을 원할 때는 Axis.vertical을 사용한다.
3. 기본 스크롤 방향은 Axis.vertical이므로, 세로 스크롤을 만들 때는 scrollDirection 설정을 사용하지 않아도 된다



&nbsp;ListView(

&nbsp;   scrollDirection: Axis.horizontal,

&nbsp;   children: List.generate(

&nbsp;	10,

&nbsp;	(index) => Container(

&nbsp;	  width: 100,

&nbsp;	  height: 100,

&nbsp;	  margin: const EdgeInsets.all(5),

&nbsp;	  color: Colors.red.withAlpha((index + 1) \* 25),

&nbsp;     ),

&nbsp;  ),

), 





###### Controller

1. Controller 옵션에 등록할 scrollController 클래스를 사용하여 스크롤을 제어합니다
2. 원하는 스크롤 위치로 이동하거나, 현재 스크롤 위치를 실시간으로 알 수 있다.
3. 보통 controller 없이 사용하지만, 무한 스크롤처럼 특정한 요구사항이 있을때 사용한다
4. 예를 들어 마지막 스크롤위치에서 다음 페이지를 불러오는 이벤트를 처리할 수 있다.





###### physics

* physics 옵션은 ListView 위젯에서 스크롤 동작을 결정하는 속성이다 이 속성을 사용하여 리스크를

&nbsp;    동작을 커스터마이즈함으로써 사용자가 앱에서 더 나은 스크롤 경험을 할 수 있다

* physics 옵션에는 다양한 효과가 있다



1. BouncingScrollPhysics: 스크롤 범위가 내부 콘텐츠 보다 큰 경우, 스크롤 끝에서 반사 효과를 제공한다. iOS의

&nbsp;    스크롤 동작과 유사하다

2\. ClampingScrollPhysics: 스크롤 범위가 내부 콘텐츠 보다 클 때, 끝에서 반사 효과 대신 스크롤을 멈춘다.

&nbsp;   안드로이드 스크롤 동작과 유사하다

3\. FixedExtentScrollPhysics: 모든 아이템이 동일한 크기를 가질 떄 사용된다. 스크롤이 균일한 단위로 이동한다

4\. NeverScrollableScrollPhysics: 스크롤이 비활성화 상태이다





##### 4.4.9 PageView 위젯

* PageView는 책이나 프레젠테이션 처럼 가로 또는 스와이프하여 일련의 페이지를 볼 수 있게해주는 위젯으로

&nbsp;    손가락으로 화면에 넘기는 것처럼 스와이프 제스처로 페이징할 수 있는 기능을 제공한다.
